# T047 - Yoga Studio

## Overview
A serene yoga studio template with zen calm design, perfect for yoga studios, wellness centers, and meditation spaces.

## Features
- **Zen Design**: Calming color palette (#8baa7e, #f7f3ee, #d4a574)
- **12 Complete Pages**: Full yoga studio website
- **Class Scheduling**: Weekly timetable
- **Instructor Profiles**: Teacher showcase
- **Wellness Blog**: Health articles
- **Gallery**: Studio photos
- **Trial Class**: Free trial booking
- **Mobile Responsive**: Hamburger menu

## Color System
| Color | Hex | Usage |
|-------|-----|-------|
| Primary | #8baa7e | Main brand, nature elements |
| Secondary | #f7f3ee | Warm backgrounds |
| Accent | #d4a574 | Highlights, CTAs |
| Dark | #3d4f3a | Headings |
| Light | #faf9f6 | Light backgrounds |

## Typography
- **Headings**: Cormorant Garamond
- **Body**: Noto Sans TC
