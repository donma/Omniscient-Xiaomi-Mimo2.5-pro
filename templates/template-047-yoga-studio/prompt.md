# Yoga Studio Template Prompt

## Template Overview
Create a serene yoga studio website with a zen calm design system (#8baa7e, #f7f3ee, #d4a574) that conveys peace and wellness.

## Design Direction
- **Style**: Serene, natural, zen, calming
- **Color Palette**: Sage green (#8baa7e), warm cream (#f7f3ee), earthy tan (#d4a574), dark green (#3d4f3a)
- **Typography**: Cormorant Garamond (headings) + Noto Sans TC (body)
- **Imagery**: Unsplash yoga/nature images, soft overlays

## Page Structure

### Homepage (index.html)
1. **Hero Section**: Yoga pose in nature with soft overlay, headline "Find Your Inner Peace"
2. **Class Types**: Vinyasa, Hatha, Yin, Restorative, Meditation
3. **Instructors**: Featured instructor spotlight
4. **Schedule**: Weekly class timetable
5. **Studio Gallery**: Photo gallery of studio spaces
6. **Wellness Blog**: Latest articles
7. **Trial Class CTA**: Free trial class invitation
8. **Contact**: Location and contact info
9. **Newsletter**: Email signup

## Technical Requirements
- Bootstrap 5 CDN
- Bootstrap Icons CDN
- Google Fonts (Cormorant Garamond, Noto Sans TC)
- Mobile hamburger menu
- FAQ accordion
- Form submission with thank you message
- Schema.org markup
- Open Graph tags
- Responsive design
