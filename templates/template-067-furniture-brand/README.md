# 木匠工坊

家具品牌網站模板

## 色彩方案

| 顏色 | 色碼 | 用途 |
|------|------|------|
| 主色調 | #f5f5f5 | 背景 |
| 輔助色 | #8b7355 | 按鈕、裝飾 |

## 字體: Montserrat

## 頁面列表

1. 首頁 (index.html)
2. 關於我們 (about.html)
3. 服務項目 (services.html)
4. 服務詳情 (service-detail.html)
5. 作品展示 (portfolio.html)
6. 客戶評價 (reviews.html)
7. 常見問題 (faq.html)
8. 預約諮詢 (booking.html)
9. 服務流程 (process.html)
10. 部落格 (blog.html)
11. 文章詳情 (blog-detail.html)
12. 聯絡我們 (contact.html)

## 技術架構

- CSS框架: Bootstrap 5.3.0
- 圖示庫: Bootstrap Icons 1.11.0
- 字體: Google Fonts
- JavaScript: Vanilla JS

## 使用方式

1. 下載模板檔案
2. 使用瀏覽器開啟 index.html 即可預覽
3. 根據需求修改內容與圖片
4. 上傳至您的網站伺服器
