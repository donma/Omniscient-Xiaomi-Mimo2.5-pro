# T003 Fashion Boutique 模板使用說明

## 模板簡介
NOIR是一個專為時尚服飾產業設計的網站模板，採用極簡奢華風格設計，適合精品服飾、設計師品牌、時尚選物店使用。

## 使用方式
1. 下載整個資料夾
2. 用瀏覽器開啟 `index.html` 即可預覽
3. 根據需求修改文字、圖片、連結

## 自訂顏色
修改 `assets/css/style.css` 頂部的CSS變數：
```css
:root {
    --primary: #1a1a1a;  /* 黑色 */
    --gold: #c9a96e;     /* 金色 */
}
```

## 功能特色
- ✅ Bootstrap 5 響應式框架
- ✅ 極簡奢華設計風格
- ✅ SEO優化
- ✅ 產品展示卡片
- ✅ Lookbook網格
