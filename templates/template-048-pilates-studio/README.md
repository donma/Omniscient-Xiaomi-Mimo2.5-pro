# T048 - Pilates Studio

## Overview
A premium boutique Pilates studio template with elegant beige design, perfect for upscale fitness studios.

## Features
- **Premium Design**: Beige and charcoal color scheme
- **12 Complete Pages**: Full studio website
- **Equipment Showcase**: Reformer and equipment display
- **Membership Plans**: Pricing tiers
- **Mobile Responsive**: Hamburger menu
