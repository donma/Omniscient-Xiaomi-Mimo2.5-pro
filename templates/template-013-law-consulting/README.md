# T013 Law Consulting

Authoritative law consulting website template with prestigious dark navy and gold design.

## Features
- Responsive design with Bootstrap 5
- Attorney profiles
- Practice areas
- Case results
- Consultation booking
- FAQ accordion

## Colors
- Primary: #1a365d (Deep Navy)
- Secondary: #c9a84c (Gold)
- Light: #f7f7f7

## Fonts
- Merriweather (Headings)
- Noto Sans TC (Chinese text)
