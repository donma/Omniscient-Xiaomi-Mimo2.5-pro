# T046 - Online Course Platform

## Overview
A modern e-learning platform template with a stunning purple gradient design system, perfect for online education businesses, course creators, and learning management systems.

## Features
- **Purple Gradient Design**: Eye-catching gradient system (#667eea → #764ba2)
- **12 Complete Pages**: All pages needed for a full course platform
- **Course Catalog**: Filterable course grid with category navigation
- **Instructor Profiles**: Detailed instructor showcase sections
- **Pricing Plans**: 3-tier pricing comparison table
- **Student Reviews**: Testimonial carousel with ratings
- **Learning Path**: Visual course progression pathway
- **FAQ Accordion**: Expandable frequently asked questions
- **Mobile Responsive**: Fully responsive with hamburger menu
- **SEO Optimized**: Meta tags, OG tags, Schema.org markup

## Color System
| Color | Hex | Usage |
|-------|-----|-------|
| Primary | #667eea | Main brand, buttons, links |
| Secondary | #764ba2 | Gradients, accents |
| Accent | #f093fb | Highlights, hover states |
| Dark | #2d1b69 | Headings, dark sections |
| Light | #f8f9ff | Backgrounds, cards |

## Typography
- **Headings**: Poppins (600, 700)
- **Body**: Poppins (300, 400, 500)

## Pages
1. `index.html` - Homepage with 8+ sections
2. `about.html` - About the platform
3. `courses.html` - Course catalog
4. `course-detail.html` - Single course page
5. `instructors.html` - Instructor profiles
6. `pricing.html` - Pricing plans
7. `blog.html` - Educational blog
8. `blog-post.html` - Single blog post
9. `testimonials.html` - Student reviews
10. `faq.html` - FAQ page
11. `contact.html` - Contact form
12. `enroll.html` - Enrollment form

## Technical Stack
- Bootstrap 5.3 CDN
- Bootstrap Icons 1.11
- Google Fonts (Poppins)
- Vanilla JavaScript
- Unsplash images

## Usage
1. Copy the template folder
2. Customize colors in CSS variables
3. Replace Unsplash images with your own
4. Update content and links
5. Deploy to your hosting
