# T004 Coffee Shop 模板使用說明

## 模板簡介
拾光咖啡是一個專為咖啡店設計的網站模板，採用溫暖手工風格設計，適合精品咖啡、甜點店、複合式空間使用。

## 使用方式
1. 下載整個資料夾
2. 用瀏覽器開啟 `index.html` 即可預覽
3. 根據需求修改文字、圖片、連結

## 自訂顏色
修改 `assets/css/style.css` 頂部的CSS變數：
```css
:root {
    --primary: #6f4e37;  /* 咖啡棕 */
    --secondary: #f4e9d9; /* 奶油色 */
}
```

## 功能特色
- ✅ Bootstrap 5 響應式框架
- ✅ 溫暖手工風格設計
- ✅ SEO優化
- ✅ 菜單展示卡片
- ✅ 空間相冊網格
