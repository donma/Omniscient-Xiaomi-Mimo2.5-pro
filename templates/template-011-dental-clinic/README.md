# T011 Dental Clinic Template

## Overview
Professional dental clinic website template featuring clean medical design, trust-building elements, and comprehensive patient information.

## Files Structure
```
T011-dental-clinic/
├── index.html              # Homepage with 8+ sections
├── about.html              # About the clinic & doctors
├── services.html           # All dental services
├── service-detail.html     # Single service detail page
├── portfolio.html          # Before/after cases
├── reviews.html            # Patient testimonials
├── faq.html                # Dental FAQ accordion
├── booking.html            # Appointment booking
├── process.html            # Treatment process
├── blog.html               # Dental health articles
├── blog-detail.html        # Single article
├── contact.html            # Contact & location
├── assets/
│   ├── css/
│   │   ├── style.css       # Main stylesheet
│   │   └── responsive.css  # Responsive rules
│   ├── js/
│   │   └── main.js         # Interactive features
│   └── img/
│       └── image-sources.md
├── template.json
├── tags.json
├── prompt.md
└── README.md
```

## CDN Dependencies
- Bootstrap 5.3.x
- Bootstrap Icons 1.11.x
- Google Fonts: Inter, Noto Sans TC

## Features
- Responsive mobile-first design
- LocalBusiness Schema (DentalClinic type)
- Appointment booking form
- FAQ accordion
- Before/after image gallery
- Insurance information section
- Emergency contact banner
- Google Maps placeholder

## Customization
Edit `assets/css/style.css` CSS variables to change colors:
```css
:root {
  --primary: #2196F3;
  --secondary: #4caf50;
  --light: #e8f4fd;
}
```
