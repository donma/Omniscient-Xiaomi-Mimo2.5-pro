# T012 Pet Grooming Template

## Overview
Warm and playful pet grooming website template featuring cute design elements, service booking, and pet care tips.

## Files Structure
```
T012-pet-grooming/
├── index.html              # Homepage with 8+ sections
├── about.html              # About us
├── services.html           # All services
├── service-detail.html     # Single service detail
├── portfolio.html          # Groomed pets gallery
├── reviews.html            # Pet parent reviews
├── faq.html                # FAQ accordion
├── booking.html            # Service booking
├── process.html            # Grooming process
├── blog.html               # Pet care tips
├── blog-detail.html        # Single article
├── contact.html            # Contact & location
├── assets/
│   ├── css/
│   │   ├── style.css       # Main stylesheet
│   │   └── responsive.css  # Responsive rules
│   ├── js/
│   │   └── main.js         # Interactive features
│   └── img/
│       └── image-sources.md
├── template.json
├── tags.json
├── prompt.md
└── README.md
```

## CDN Dependencies
- Bootstrap 5.3.x
- Bootstrap Icons 1.11.x
- Google Fonts: Quicksand, Noto Sans TC

## Features
- Responsive mobile-first design
- LocalBusiness Schema (PetGrooming type)
- Service booking form
- FAQ accordion
- Pet gallery with filtering
- Pricing packages
- Pet care tips section

## Customization
Edit `assets/css/style.css` CSS variables to change colors:
```css
:root {
  --primary: #ff9a9e;
  --secondary: #ffecd2;
  --accent: #fcb69f;
}
```
