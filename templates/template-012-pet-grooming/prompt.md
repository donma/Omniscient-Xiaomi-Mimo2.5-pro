# T012 Pet Grooming Template Generation Prompt

## Template Overview
Create a warm and playful pet grooming website that feels friendly and approachable while maintaining professionalism.

## Design Direction
- **Style**: Warm, friendly, playful but professional
- **Color Palette**: Warm peach (#ffecd2), Soft coral (#fcb69f), Dark text (#4a4a4a), Pink accent (#ff9a9e)
- **Typography**: Quicksand for headings, Noto Sans TC for Chinese text
- **Layout**: Rounded corners, soft shadows, playful decorations

## Hero Section
Cute pet photo with playful overlay, paw print decorations, and a warm welcome message.

## Required Sections (Homepage)
1. Hero with playful pet image
2. Services Grid (bath, trim, nail, teeth cleaning)
3. Gallery of Groomed Pets
4. Pricing Packages
5. Booking CTA
6. Pet Care Tips
7. Reviews from Pet Parents
8. Location
9. Final CTA

## Card Design
Rounded corners, soft shadows, peach backgrounds, cute icons.

## Navigation
Playful, rounded buttons, warm colors, paw print logo.

## Special Requirements
- LocalBusiness Schema for pet grooming
- Paw print decorative elements
- Before/after pet grooming gallery
- Cute pet emoji/icons usage
- Pet care blog section
- All images from Unsplash pet collections
